# Underess
#### Android: PIN解锁

- **原理：使用 `EditText` 控件，通过监听 Return 是否被点击，来接受输入框内的密码。使用 `SharedPreferences` 对密码进行持续化存储。**

- **学习内容：`TextView` `EditText` `Button` `SharedPreferences` `ObjectAnimator` `View的setOnTouchListener` `ViewGroup`**

- **难度：新手。**
